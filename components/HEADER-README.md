# Новая шапка сайта (Header Component)

## Обзор

Компонент `Header` реализует двухуровневую навигацию согласно дизайну из Figma с точными размерами и стилями.

## Расположение файла

```
components/Header.tsx
```

## Дизайн и стили

### Основные параметры:
- **Общая высота:** 124px (68px основная + 56px вторичная)
- **Позиционирование:** Fixed, z-index: 50
- **Фон:** Белый (#FFFFFF)
- **Шрифт:** Onest (все веса подключены в globals.css)

### Первая строка (Main Header) - 68px:
- **Логотип:** Изображение из `/images/mainlogo.png` (120x40px, приоритетная загрузка)
- **Поиск:** Ширина 443px, высота 43px, бордер #EDEDED, радиус 12px
- **Навигация:** "Ustozlar uchun", "O'quvchilar uchun"
- **Кнопка входа:** "Kirish" - фон #1761FF, hover #1451dd
- **Отступы:** 320px (адаптивные: 80px на sm, 160px на lg, 320px на xl)
- **Тень:** 0px 0px 2px rgba(0,0,0,0.15)

### Вторая строка (Secondary Navigation) - 56px:
- **Категории:** Darsliklar, Resurslar, Tarqatma materiallar, Taqdimotlar, Kurslar
- **Мега-меню:** Появляется при hover на "Darsliklar" и "Resurslar"
- **Ширина мега-меню:** На всю ширину контейнера (с учетом padding)
- **Структура:** 3-колоночная сетка с заголовками секций
- **Отступ сверху:** Бордер 1px #EDEDED
- **Тень:** 0px 1px 1px rgba(0,0,0,0.05)
- **Правый элемент:** "Element" (заполнитель)

## Использование

### Базовое использование:

```tsx
import Header from '@/components/Header';

export default function YourPage() {
  return (
    <div>
      <Header />
      {/* Добавьте отступ сверху для контента */}
      <div className="mt-[124px]">
        {/* Ваш контент здесь */}
      </div>
    </div>
  );
}
```

### С другим layout:

```tsx
import Header from '@/components/Header';

export default function CustomLayout({ children }) {
  return (
    <div className="min-h-screen">
      <Header />
      <main className="mt-[124px]">
        {children}
      </main>
    </div>
  );
}
```

## Интерактивные элементы

### Поиск:
- Состояние управляется через `useState`
- Placeholder: "Izlash"
- Цвет placeholder: #BDC3E3

### Мега-меню:
- **Активация:** При наведении на "Darsliklar" или "Resurslar"
- **Деактивация:** При уходе курсора с меню
- **Структура:** 3 колонки с секциями
- **Содержимое:** Каждая секция имеет:
  - Заголовок (uppercase, tracking-wider)
  - Список элементов с названием и описанием
- **Стили:**
  - Фон: белый (#FFFFFF)
  - Border top: 1px #EDEDED
  - Shadow: большая тень для выделения
  - Padding: совпадает с основным контейнером
  - Hover на элементах: название становится #1761FF

### Настройка мега-меню:
Данные для мега-меню находятся в константе `megaMenus` внутри компонента. Структура:

```tsx
const megaMenus = {
  textbooks: [
    {
      title: 'Название секции',
      items: [
        {
          name: 'Название элемента',
          href: '/ссылка',
          description: 'Описание'
        }
      ]
    }
  ]
};
```

### Ссылки:
- Hover эффект: цвет меняется на #1761FF
- Transition: все переходы плавные (transition-colors)

### Кнопка "Kirish":
- Базовый цвет: #1761FF
- Hover цвет: #1451dd
- Переход на /auth/login

## Адаптивность

Компонент адаптивен с помощью Tailwind breakpoints:

- **Mobile (<1024px):** padding 80px (px-20)
- **Large (1024px-1280px):** padding 160px (lg:px-40)
- **XLarge (>1280px):** padding 320px (xl:px-80)

## Цветовая палитра

```css

--border-gray: #EDEDED
--placeholder-gray: #BDC3E3
--text-black: #000000
--bg-white: #FFFFFF
```

## Технические детали

### Зависимости:
- Next.js Link (для навигации)
- React useState (для состояния поиска)
- Tailwind CSS (для стилей)

### Производительность:
- Client-side компонент ('use client')
- Минимальные re-renders (только при изменении поиска)
- Оптимизированные hover эффекты через CSS transitions

## Интеграция с существующим проектом

Компонент уже интегрирован в:
- ✅ `app/page.tsx` - Главная страница
- ✅ `app/demo-header/page.tsx` - Демо страница

Для добавления на другие страницы просто импортируйте компонент и добавьте отступ `mt-[140px]` к основному контенту.

## Кастомизация

Для изменения внешнего вида отредактируйте файл `components/Header.tsx`:

1. **Логотип:** Измените текст "edubaza" или добавьте изображение
2. **Навигация:** Добавьте/удалите ссылки в массиве
3. **Цвета:** Измените цвета в className (или добавьте в globals.css)
4. **Размеры:** Измените значения h-[84px], h-[56px] и др.

## Шрифт Onest

Шрифт Onest уже подключен в проекте:
- Расположение: `public/fonts/Onest-*.ttf`
- Подключение: `app/globals.css` (строки 5-36)
- Использование: `font-['Onest']` в компоненте
